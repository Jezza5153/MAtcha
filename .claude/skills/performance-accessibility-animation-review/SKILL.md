# Performance Accessibility Animation Review

Use this skill when reviewing animated frontend code.

## Check for
- prefers-reduced-motion support
- Lazy-loaded images/video
- Compressed video formats
- No huge autoplay videos on mobile unless optimized
- Proper alt text
- Keyboard navigation
- Button focus states
- Lighthouse performance risk
- ScrollTrigger cleanup
- Mobile scroll bugs
- No text trapped inside images
