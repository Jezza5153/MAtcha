# Stitch UI Brief Builder

Use this skill when preparing prompts for Google Stitch.

## Output required
Create a visual design prompt that includes:
- Brand mood
- Target customer
- Page sections
- Color palette
- Typography direction
- Image style
- Animation notes
- Mobile-first layout rules
- Ecommerce conversion requirements

## Rules
- Do not ask Stitch for final production code.
- Stitch is used for visual exploration and layout direction.
- Designs must be exportable into a production React/Next.js implementation.
