# Ecommerce Conversion Guardrails

Use this skill when reviewing ecommerce UX.

## Rules
- The site must sell product, not only impress.
- Hero must explain what is sold within 5 seconds.
- CTA must appear above the fold.
- Product benefits must be concrete: taste, origin, grade, preparation, shipping.
- Do not hide product cards behind too much storytelling.
- Checkout path must be obvious.
- Mobile product purchase must take minimal taps.
- Add trust: reviews, origin, organic/lab/certification if available, shipping, return policy.
