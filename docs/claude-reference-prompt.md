# Claude reference-driven build prompt

Use this when the owner ships videos / GIFs / images of motion they like. Upload references first, then paste the prompt.

---

I will upload reference videos/GIFs/images.

Analyze them first and extract:
1. What animation behavior is happening
2. What timing/pacing makes it feel premium
3. What should be copied
4. What should not be copied
5. How to recreate the same feeling using GSAP ScrollTrigger and React

Do not copy the brand or design directly.
Use the references only for motion language and layout inspiration.

After analysis, create a matcha ecommerce homepage animation plan with:
- Section-by-section scroll behavior
- Pinning behavior
- Timeline states
- Required assets
- Mobile fallback
- Performance risks
- Implementation steps
