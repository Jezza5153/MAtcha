# Stitch design prompt

Paste this into Google Stitch to generate visual direction. Use the output for layout exploration only; production code lives in this repo.

---

Create a premium ecommerce landing page design for a matcha brand.

The site should feel calm, expensive, natural, and Japanese-inspired, but not cliché. Avoid generic wellness design, fake zen symbols, cheap green gradients, and childish animations.

Brand mood:
- Premium ceremonial matcha
- Quiet luxury
- Organic texture
- Modern editorial layout
- Warm cream background
- Deep matcha green
- Soft stone / ceramic details
- Minimal but sensory

Target customer:
People who care about quality matcha, better daily rituals, health, taste, and premium ingredients. They may buy ceremonial matcha, everyday matcha, or a starter kit.

Page sections:
1. Hero with product pack, matcha powder texture, short headline, CTA
2. Origin story with tea fields / shade-grown leaves
3. Stone-ground quality section
4. Whisking ritual section
5. Taste notes section: creamy, umami, smooth, not bitter
6. Product cards: Ceremonial Matcha, Everyday Matcha, Starter Kit
7. Reviews / trust
8. Final buy section

Visual rules:
- Lots of whitespace
- Big editorial typography
- Premium product photography
- Soft natural shadows
- No clutter
- Mobile-first
- Clear buy buttons

Animation direction:
Design the layout as if it will later be built with scroll animations:
- Product pinned in center during intro
- Background changes from powder to leaves to whisking
- Text fades slowly as user scrolls
- Product cards reveal smoothly
- Motion should be calm, premium, not playful

Create a homepage concept with desktop and mobile layout.
